Demosgg, a full stack Rocket League Esports website
Welcome to my Rocket League esports website, a full-stack platform designed to bring you comprehensive and up-to-date information about the world of Rocket League. Designed and Programmed by me. Built using Flask, HTML, CSS, and JavaScript, this website harnesses the power of the zsr.octane.gg API to deliver a rich and dynamic user experience.
This project showcases my ability to develop a full-stack web application that integrates multiple technologies to create a user-friendly and informative platform. Whether you're a casual fan or a dedicated esports enthusiast, this website has everything you need to stay informed about the Rocket League competitive scene.
  1. The standout feature is the highly efficient global search function. Users can quickly find players, teams, and events, each neatly displayed in their dedicated sections. This allows for a seamless and intuitive search experience.
  2. Explore individual pages for players, teams, events, and matches. Each page is packed with detailed statistics and information, ensuring you have all the data you need at your fingertips.
  3. For those seeking more specific information, the website offers individual search pages for players, teams, and events. These pages feature infinite scrolling and region filters, allowing for effortless browsing through extensive lists.
  4. The entire website is fully responsive, providing an optimal viewing experience on mobile devices. My custom UI ensures that the site looks and works great on any screen size.
